# Shopify
Simple E-commerce website created using Django.

## Technologies Used
**Front-end:**
- HTML 5
- CSS 3
- jQuery 3.5.1
- Bootstrap 4.5.2

**Back-end:**
- Django 3.1.5

**Database:**
- PostgreSQL 13

**Web Server:**
- Apache 2.4

**Cloud Platform:**
- Amazon Web Services Windows Server 2019
